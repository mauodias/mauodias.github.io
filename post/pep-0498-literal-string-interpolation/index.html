PEP 0498: Literal String Interpolation

12 Mar 2019 on one pep a week pep python string interpolation development

Being an awesome text manipulation language as it is, Python has quite a few ways to interpolate strings. Some of them can be seen below:

  • %-formatting

    word = "awesome"
print("This place is %s, isn't it?" % word)
# This place is awesome, isn't it?

  • str.format()

    word = "awesome"
print("Python is an {} programming language!".format(word))
# Python is an awesome programming language!

  • string.Template
